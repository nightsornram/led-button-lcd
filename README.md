# led-button-lcd

**ต้องกดปุ่มค้างเอาไว้ประมาณ 1 วิ**

LCD :
GND >> GND , VCC >> VBUS , SDA >> GPO , SCL >> GP1

RGB Module : 
GND >> GND , R >> GP2 , G >> GP3 , B >> GP4

Button1 :
GND >> GND , PIN >> GP7

Button2 :
GND >> GND , PIN >> GP5

Button3 :
GND >> GND , PIN >> GP6
